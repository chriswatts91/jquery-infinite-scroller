jQuery Infinite Scroller

See readme on Github for usage instructions.

See demos directory in repo for the code behind these demos.

Demos

Simple slider

        $("#example1").infiniteScroller();

Scrolling right without hover

        $( "#example2" ).infiniteScroller( {
            scrollOnMouseOver: false,
            direction: "right"
        } );

Maintain speed on mouseout

        $( "#example3" ).infiniteScroller( {
            resetOnMouseOut: false,
            maxSpeed: 50
        } );
